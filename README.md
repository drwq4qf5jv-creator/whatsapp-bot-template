# قالب بوت واتساب بسيط (Node.js + Baileys)
## ملخص سريع بالعربي
هذا مشروع بسيط يمكنك رفعه إلى GitHub ثم استيراده في **Replit** من الآيفون وتشغيله. بعد التشغيل سيظهر لك **QR Code** في وحدة التحكم (Console) لربط بوتك بحساب واتساب.

### خطوات سريعة (آيفون - بدون حاسوب)
1. افتح تطبيق **GitHub** أو الموقع github.com وسجّل دخولك (أو أنشئ حساب جديد).
2. أنشئ Repository جديد باسم `whatsapp-bot-template` — أو أي اسم تريده.
3. ارفع الملفات الموجودة هنا (`index.js`, `package.json`, `README.md`) إلى الـRepository. (في تطبيق GitHub: اضغط Add file → Upload files).
4. افتح **Replit** على متصفح الآيفون.
5. اختر **Create** → **Import from GitHub** وأدخل رابط الـRepository الذي أنشأته.
6. بعد الاستيراد افتح علامة التبويب **Shell** أو **Console** في Replit.
7. شغّل:
   ```
   npm install
   npm start
   ```
   أو فقط اضغط Run إذا ظهر زر Run.
8. ستظهر صورة QR داخل Console — افتح **WhatsApp** → **الإعدادات** → **الأجهزة المرتبطة** → **ربط جهاز** ثم امسح الـQR.
9. الآن البوت متصل. اكتب رسالة `ping` إلى أي رقم مرتبط بالبوت وستستلم `pong`.

### ملاحظات مهمة
- Replit قد يوقف التطبيق بعد فترة؛ استخدام خدمة Keep-Alive مثل UptimeRobot يساعد على إبقاء البوت متصلًا.
- هذا قالب بسيط للتعلم. أنصحك بعد التشغيل أن تكتب هنا وسأرشدك خطوة بخطوة لتخصيص الأوامر وإضافة ميزات.
- إذا واجهت خطأ 404 في GitHub عند استيراد روابط جاهزة، تأكد أنك أنشأت الريبو الخاص بك ورفعت الملفات إليه.

### الملفات في القالب
- `index.js` : ملف التشغيل الرئيسي.
- `package.json` : تعريف المشروع والحزم المطلوبة.
- `auth_info.json` : سينشأ بعد أول تشغيل ويحفظ بيانات الجلسة تلقائيًا.

### مساعدة لاحقة
لو تريد، أقدر:
- أرفع هذا القالب على حساب GitHub خاص بك (أحتاج تصريحك للدخول إلى حسابك على GitHub — لكن لا أوصي بمشاركة الباسورد؛ الأفضل أن تتبع الخطوات وتطلب مني أي مساعدة أثناء الرفع).
- أو أجهز ملف ZIP تحميل مباشر أرسله لك لتحمّله وترفعه بنفسك.

اكتب لي: "جهّز لي الريبو" وسأجهّز ملفات ZIP أو أشرح خطوة رفعها خطوة بخطوة على الآيفون.
